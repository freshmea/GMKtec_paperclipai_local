# Code Examples for Robotics Education

This directory contains curated source code examples designed for educational purposes in robotics.

## Contents

### ROS 2 Examples
- `ros2_examples/minimal_publisher.py`: A basic ROS 2 Python node that publishes a "Hello World" string to a topic.
- `ros2_examples/subscribers/minimal_subscriber.py`: A basic ROS 2 Python node that subscribes to a topic and prints received messages.

### OpenCV Examples
- `opencv_examples/basic_camera_grayscale.py`: A script to capture video from a camera and display it in grayscale using OpenCV.
- `opencv_examples/detection/face_detection.py`: A script using Haar Cascades to detect faces in a real-time video stream.

### Python Robotics Examples
- `python_robotics_examples/basic_kinematics.py`: A simple implementation of forward kinematics for a 2-DOF planar robotic arm.

## Usage
Ensure you have the necessary libraries installed:
- `rclpy` (for ROS 2)
- `opencv-python` (for OpenCV)
- `numpy` (for Python robotics)
