# AI and Advanced Robotics

## Computer Vision
Computer vision enables robots to interpret and understand visual information from their surroundings, which is critical for autonomy.

- **Core Tasks**: Object Detection, Image Segmentation, Feature Extraction, Optical Flow, Depth Estimation.
- **Key Techniques**: CNNs (Convolutional Neural Networks), YOLO (You Only Look Once), Vision Transformers (ViT), Visual SLAM (Simultaneous Localization and Mapping).
- **Applications**: Autonomous Navigation, Object Manipulation, Human-Robot Interaction, Quality Inspection.

## Algorithms and Control
Essential for movement, localization, and maintaining stability in dynamic environments.

- **Navigation**: A*, Dijkstra's, RRT (Rapidly-exploring Random Trees), D*.
- **Localization & Mapping**: SLAM, Extended Kalman Filter (EKF), Particle Filters.
- **Control**: PID (Proportional-Integral-Derivative), Model Predictive Control (MPC), Reinforcement Learning (RL) for adaptive control.

## Sensors and Actuators
The hardware interface between the robot and its environment, providing data and enabling physical action.

- **Sensors**: 
    - **Distance/Proximity**: Ultrasonic, IR, Lidar.
    - **Orientation/Motion**: IMU (Inertial Measurement Unit), Gyroscopes, Accelerometers.
    - **Visual**: Cameras (RGB, Depth/Stereo).
    - **Tactile**: Touch sensors, Pressure sensors.
- **Actuators**: 
    - **Electric**: DC Motors, Servo Motors, Stepper Motors.
    - **Fluidic**: Pneumatic and Hydraulic actuators.

## Summary of Advanced Robotics Components

| Component | Primary Function | Key Examples |
| :--- | :--- | :--- |
| **Perception** | Understanding surroundings | Computer Vision, Lidar, Sensors |
| **Planning** | Deciding on actions | A*, RRT, Pathfinding algorithms |
| **Control** | Executing physical movement | PID, MPC, Motor drivers |
| **Actuation** | Physical interaction | Servos, DC Motors, Pneumatics |
