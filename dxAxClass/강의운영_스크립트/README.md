# DXAX 시간별 강의운영 스크립트

이 폴더는 `교안패키지`의 일차별 문서를 실제 강의 진행 단위인 `교시` 수준으로 세분화한 운영 런북이다. 각 교시는 `스크립트`, `참조할 내용`, `수업 팁`의 세 요소로 고정해 강의 흐름이 흔들리지 않게 설계했다.

## 폴더 구성

- `1주차_시간별_스크립트.md`: DX·AX 기초 이해, 업무 진단, 문제 정의, 데이터 요구사항
- `2주차_시간별_스크립트.md`: 프롬프트 설계, 문서 작성, 요약, 데이터 해석
- `3주차_시간별_스크립트.md`: 자동화 구조 설계, 협업 적용, 효율화, 변화관리
- `4주차_시간별_스크립트.md`: 이미지·영상 생성형 AI, n8n, 실행 로드맵

## 사용 원칙

- 매 교시 시작 1분은 `전 시간 산출물 호출`에 쓴다.
- 오전 교시는 개념 정렬, 오후 교시는 적용과 산출물 생산, 8교시는 리뷰와 다음 날 입력값 저장으로 운영한다.
- 실습 교시는 설명을 줄이고 `목표 -> 제출물 -> 시간 -> 검수 기준` 순으로만 말한다.
- 5일 단위로 주간 서사가 닫히도록 설계했으므로, 일차를 건너뛰어 사용하지 않는 것을 권장한다.

## 주간 연결 구조

- 1주차: 업무를 보는 관점을 바꾸고 문제와 데이터 구조를 잡는다.
- 2주차: 1주차 문서를 입력값으로 삼아 재현 가능한 결과물을 만든다.
- 3주차: 2주차 결과물을 자동화와 협업 흐름으로 전환한다.
- 4주차: 확장 도구를 익히고 수료 직후 실행할 로드맵으로 수렴한다.

## 권장 사용 순서

1. 수업 전 `교안패키지/진행자_노트.md`로 준비물과 리스크를 확인한다.
2. 수업 중 해당 주차의 시간별 스크립트 파일을 열어 교시 단위 멘트와 운영 팁을 사용한다.
3. 필요할 때 `교안패키지/발표자_스크립트.md`로 일차 전체 메시지를 다시 맞춘다.
4. 8교시 종료 전에 당일 산출물을 다음 교시 입력값 이름으로 저장한다.
